== Theme Install And Set Up ==

* The theme can be installed just like anyother theme - either via FTP
* or by uploading it via Admin >> Appearance >> Install Themes

* Slider set up is simple and quick - simply visit the theme customizer
* and under the slider options select to enable the slider, select the
* desired category, set the number of slides to be show and save.

* Please note: To initiate the slider effect you must have at least two 
* Posts published within the selected category. Remeber to set your desired.

* Home Intro: This is an introduction message that appears at the top of the content
* and just below the slider on the home page - works on both the index.php and the
* Custom Home Page template when set as a static page.

* Custom Home - Content Options: This setting allows the showing of content entered
* in the editor when creating a page in conjunction with the Custom Home page template
* to be used as a static page.

* Showcase: This is a widgetized area just above the content on the home page - the
* widget area is dynamic in that it will accommodate between 1 and 5 widgets.

* The rest of the set up is per any other theme - simply post your content.
